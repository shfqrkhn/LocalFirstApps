# THE COMPLETE STRENGTH PROTOCOL

**Full-Body Strength • Time-Efficient • Evidence-Based**

---

## 1. PROTOCOL OVERVIEW

### Design Principles
This protocol uses standing, bench-supported, and equipment-based movements exclusively. All exercises can be performed without floor contact, allowing consistent execution in any gym environment.

### Key Parameters

| Parameter | Value | Notes |
|-----------|-------|-------|
| **Frequency** | 3 sessions per week | App calculates week number based on this |
| **Minimum Rest** | 24 hours between sessions | App enforces this minimum |
| **Recommended Rest** | 48–72 hours between sessions | Optimal recovery window |
| **Inactivity Warning** | 7 days (168 hours) | App warns if exceeded |
| **Total Exercises** | 9 movements | All tracked by app |
| **Sets Per Exercise** | 3 sets | Fixed for all exercises |
| **Rest Between Sets** | 90 seconds | Default timer in app |

### Session Phases

The app divides each workout into 4 phases:

| Phase | Duration | Description |
|-------|----------|-------------|
| 1. Warmup | 5 min | 5-movement activation circuit |
| 2. Lifting | 45 min | 9 strength exercises with rest periods |
| 3. Cardio | 5 min | Zone 2 cardiovascular work |
| 4. Decompression | 3 min | Spinal traction and breathing |
| **Total** | **58 min** | Approximate session length |

### Muscle Coverage

| Movement | Sets × Reps | Primary Targets | Secondary Targets |
|----------|-------------|-----------------|-------------------|
| Hinge | 3 × 8 | Hamstrings, Glutes, Erectors | Grip, Core |
| Knee | 3 × 10 | Quadriceps, Glutes | Core, Adductors |
| Horizontal Push | 3 × 10 | Pectorals, Anterior Deltoids | Triceps |
| Incline Push | 3 × 10 | Upper Pectorals, Anterior Deltoids | Triceps |

## 2. PROGRESSION SYSTEM

### A. Recovery Assessment

Assess before each session. The app enforces this check before allowing workout to proceed.

| State | Criteria | Weight Multiplier | Action |
|-------|----------|-------------------|--------|
| **Green** | 7+ hours sleep, no acute pain or illness | × 1.0 (100%) | Train at programmed weights |
| **Yellow** | 5–6 hours sleep, minor fatigue or stiffness | × 0.9 (90%) | All weights reduced by 10% |
| **Red** | <5 hours sleep, acute pain, or illness | × 0 (0%) | No lifting; light walk only |

### B. Initial Calibration (First Session)

For each exercise on Day 1:
1. Select a weight where you can complete 12 reps with good form
2. You should have approximately 2 reps remaining in reserve (RPE 8)
3. Stop at the prescribed rep count for that exercise
4. This becomes your starting working weight

### C. Weight Progression

| Outcome | Definition | Next Session |
|---------|------------|--------------|
| **Success** | Completed all 3 sets at prescribed reps with controlled form | Add 5 lbs |
| **Failure** | Missed reps or form breakdown | Repeat same weight |
| **Stall** | Same weight failed for 3 consecutive sessions | Reduce weight by 10% |

**App Behavior:**
- Weight increment on success: **+5 lbs**
- Weight adjustment UI step: **±2.5 lbs** (allows fine-tuning)
- Stall detection threshold: **3 consecutive failures**
- Stall weight reduction: **×0.9 (10% reduction)**

### D. Scheduled Deload

| Parameter | Value |
|-----------|-------|
| Frequency | Every 6th week |
| Weight reduction | 60% of working weight |
| Trigger | Automatic, regardless of performance |

The app prompts for deload when the 6th week is reached.

### E. Missed Sessions

| Scenario | Protocol |
|----------|----------|
| Missed 1 session | Resume next available day (maintain 24hr minimum gap) |
| Missed 2+ sessions | Resume schedule; reduce weights 10% for first session |
| Illness with fever | Full rest until 24 hours fever-free; reduce weights 20% |
| Localized injury | Skip aggravating movement only; perform remaining exercises |
| 7+ days inactive | App displays warning; consider 10% weight reduction |

## 4. PHASE 2: STRENGTH TRAINING (45 MIN)

### Execution Parameters

| Parameter | Value | Notes |
|-----------|-------|-------|
| Tempo | 3-1 | 3 seconds eccentric (lowering), 1 second concentric (lifting) |
| Rest between sets | 90 seconds | App provides countdown timer with skip option |
| Sets per exercise | 3 | Fixed for all exercises |
| Breathing | Exhale on exertion | Inhale during eccentric/lowering phase |

### Time Calculation Per Exercise
- Working time: 3 sets × ~40 seconds = 2 minutes
- Rest time: 2 rest periods × 90 seconds = 3 minutes
- **Total per exercise: ~5 minutes**
- **9 exercises × 5 minutes = 45 minutes**

## 8. APP FEATURES

### Flexx Files PWA

This protocol is fully integrated with **Flexx Files**, an offline-capable progressive web app.

### Core Features

| Feature | Description |
|---------|-------------|
| Offline Operation | Works without internet; all data stored locally |
| Recovery Enforcement | Must select Green/Yellow/Red before workout begins |
| Weight Tracking | Records weight used for each exercise each session |
| Auto-Progression | Calculates +5 lb suggestion on successful completion |
| Stall Detection | Flags exercise after 3 consecutive failures at same weight |
| Deload Prompts | Reminds user at week 6 to use 60% weights |
| Rest Timer | 90-second countdown between sets with skip option |
| Cardio Timer | 5-minute (300s) countdown for cardio phase |
| Form Videos | Direct links to technique videos during workout |
| Plate Calculator | Shows exact plate loading for barbell weights |

### Data Features

| Feature | Value |
|---------|-------|
| Auto-Save | Draft saved every 30 seconds during active workout |
| Auto-Export Prompt | Triggered every 5 completed sessions |
| Export Format | JSON file containing all session data |
| Import Limit | Maximum 10 MB file size |
| Session Recovery | Prompts to restore unsaved session on app restart |

### Installation

1. Open the app URL in a mobile browser (Chrome, Safari, Firefox)
2. Select "Add to Home Screen" from browser menu
3. Launch from home screen for standalone experience
4. App works fully offline after initial load

## 10. QUICK REFERENCE CARD

### Progression Rules
- **Success (all reps complete):** +5 lbs next session
- **Failure (missed reps):** Same weight next session
- **Stall (3× same failure):** Reduce 10%
- **Deload (week 6):** Use 60% of working weight

### Recovery Multipliers
- **Green:** × 1.0 (full weight)
- **Yellow:** × 0.9 (reduce 10%)
- **Red:** × 0 (no lifting)
